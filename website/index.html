常见问题

你可能想知道

我的数据会离开我的电脑吗？

文件、对话、记忆、生成的产物与工作流状态都保存在本地。如果你使用 Rapid-MLX、Ollama 或其他本地端点，模型请求不会离开你的机器。如果你选择云端模型，提示词与相关上下文会从你的桌面直接发送到你选择的模型提供方。

我需要一个 Talos 账号吗？

不需要。Talos 不要求账号、登录、计费、充值、团队工作区或托管后端。使用云端提供方时，需要你自己的 API Key 或现有订阅。

Talos 和 ChatGPT、Claude.ai 有什么不同？

Talos 运行在你的桌面上，围绕本地文件、产物、工具、权限与工作流连续性设计。网页聊天产品是很好的助手；Talos 更接近一个能检查文件、使用工具、并把长期工作绑定到你机器上的本地 Agent 工作台。

可以离线使用吗？

可以。在 Apple Silicon 上使用 Rapid-MLX 加载 MLX 模型；在 macOS、Windows 或 Linux 上安装 Ollama 并下载模型，Talos 即可在不调用云端模型的情况下运行。

远程访问是怎么工作的？

在设置中启用远程访问，扫描二维码，打开移动端网页客户端。Talos 通过 Cloudflare 隧道与基于 Token 的认证连接，无需端口转发。
